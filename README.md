## 📌 よくある質問（FAQ）

### Q. アプリにログインできません
- 最新バージョンにアップデートして、もう一度お試しください。
- それでも解決しない場合は、下記のサポート窓口へご連絡ください。

### Q. バグを見つけました
- お手数ですが、発生状況やスクリーンショットとともにご連絡ください。

### Q. サブスクリプションが反映されません
- 一度アプリを再起動してください。
- Apple ID で購入履歴があるかご確認ください。
- それでも反映されない場合はサポートまで。
